# Dashboard de clima e tempo

Este projeto é um dashboard de clima e tempo. Ele conta com uma interface visual onde você pode acompanhar dados climáticos detalhados, visualizar o prognóstico por hora e o estendido para sete dias, receber alertas urgentes e acessar rapidamente o clima de capitais ou cidades favoritas.


## 🛠 Tecnologias
Java, JavaFX
